# Gridbound

Tactical roguelike with turn-based grid combat and unit progression.

## Concept
Chess meets roguelike. Each battle is a tactical puzzle. Build your roster across runs.

## Tech
- Vanilla JS + Canvas
- No external dependencies

## Scope
- 6-8 swarm sessions
- Strategy-focused

## Universe Link
Part of shared CodeOS universe. Eventual cross-game unlocks with Abyssal and Eidolons.
