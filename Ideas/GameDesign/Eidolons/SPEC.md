# Eidolons

Visual showcase game with heavy particle effects and stylized combat.

## Concept
Beautiful action game where visuals ARE the gameplay. Hyper-stylized, shader-heavy, particle everywhere.

## Tech
- PixiJS 7
- Custom shaders

## Scope
- 4-6 swarm sessions
- Visual focus over complexity

## Universe Link
Part of shared CodeOS universe. Eventual cross-game unlocks with Abyssal and Gridbound.
