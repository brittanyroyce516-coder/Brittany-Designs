# Abyssal

Roguelike dungeon crawler. Procedural floors, combat, items, enemies.

## Concept
Dive into procedurally generated abyssal depths. Each run different, permanent progression across runs.

## Tech
- Vanilla JS + Canvas
- No external dependencies

## Scope
- 8-10 swarm sessions
- Multi-session project

## Universe Link
Part of shared CodeOS universe. Eventual cross-game unlocks with Eidolons and Gridbound.
